    关于在html中使用transform+贝塞尔曲线实现下拉刷新这件事

    西格 7/31/2021
    每日一言

    也许你遇到一个很适合自己的人，但是错过了就是错过了。

    来自 西格的胡思乱想 (opens new window)

    # 前言

    在昨天做毕业设计的时候用到了一个下拉刷新的组件,怀着求学的心态，研究了一下transform和transition，基本实现了下拉的动作，但是维持刷新的状态保持就需要用到贝塞尔曲线，于是自己动手写了一个简单的下拉刷新。

    # 控制下拉

    在控制下拉中就需要使用到HTML5的API来监听html的状态是否在下拉

    给dom对象增加三个监听事件绑定到touch事件上

        document.addEventListener("touchmove", touch, false)
    document.addEventListener("touchstart", touch, false)
    document.addEventListener("touchend", touch, false)

    在touch事件中判断动作是触摸(touchmove)还是滑动(touchstart)或是离开屏幕(touchend)

        var touchStart = 0;// 记录起始点
    let container = document.querySelector("#container")//找到需要添加刷新的组件
    function touch(event) {
        var event = event || window.event;
        // 记录手指放下去的点
        if (event.type == "touchstart") {
            touchStart = event.changedTouches[0].pageY;// 记录起始点
        }
        // 根据手指的动作显示长度
        if (event.type == "touchmove") {
            let length = (event.changedTouches[0].clientY - touchStart).toFixed(0);
            
            container.style.transition = "transform 0.1s linear 0s"
            if (length <= 80 && length > 0) {
                container.innerHTML = "下拉刷新"
                container.style.transform = `translateY(${length}px)`
            }else if(length >= 80 && length <= 430){
                container.style.transform = `translateY(${80 + (length - 80) / 5}px)`
                container.innerHTML = "松手刷新"
            }else if (length >= 430) {
                container.style.transform = `translateY(${150}px)`
            } else{
                container.style.transform = `none`
            }
        }
        if (event.type == "touchend") {
            // 在这里手指放手后执行刷新的事件并隐藏控件
            container.innerHTML = "下拉刷新"
            container.style.transform = `translateY(${0}px)`
            container.style.transition = "transform 0.3s cubic-bezier(0.19, 1.64, 0.42, 0.72) 0s"
            
        }
    }

    通过这次的上手实践中，对贝尔塞曲线做刷新有了一定的了解，同时对刷新的实现也有了一些想法。 万事并不难，只是你觉得难，所以没有了解决的信心
